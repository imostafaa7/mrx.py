💀 MRX - Automated Recon & Vulnerability Scanning Workflow





























أداة MRX هي إطار عمل (Framework) متكامل مكتوب بلغة بايثون، مصمم لأتمتة عمليات الاستطلاع (Reconnaissance) وفحص الثغرات الأمنية بشكل شامل ودقيق. تجمع الأداة بين أقوى أدوات المجتمع الأمني في واجهة واحدة موحدة مع نظام تثبيت آلي ومؤثرات بصرية جذابة.




🚀 المميزات الرئيسية

•
التثبيت الآلي (Auto-Installer): لا داعي للقلق بشأن الأدوات المفقودة؛ تقوم MRX بفحص وتثبيت كافة المتطلبات (Go, Subfinder, Nuclei, etc) تلقائياً.

•
سير عمل متسلسل (15 مرحلة): تبدأ من اكتشاف النطاقات الفرعية وتصل إلى فحص ثغرات SQLi و XSS و SSRF.

•
واجهة رسومية CLI: واجهة مستخدم ملونة باستخدام Colorama ورسوم ASCII باستخدام Pyfiglet لتجربة اختراق احترافية.

•
تحليل الروابط التاريخية: سحب الروابط من Wayback Machine و GAU لتحليل شامل للمسارات القديمة.

•
فحص JS Endpoints: استخراج الروابط والمسارات الحساسة من ملفات الجافا سكريبت.

•
نظام تقارير ذكي: ملخص نهائي يوضح عدد النتائج في كل ملف لسهولة المراجعة.

📥 التثبيت والتشغيل

المتطلبات الأساسية

تأكد من وجود Python 3 و pip على نظامك (يفضل نظام Kali Linux أو Ubuntu).

خطوات التشغيل

1.
قم بتحميل سكربت mrx.py.

2.
قم بتشغيل الأداة مباشرة (ستقوم هي بتثبيت الباقي):

Bash


python3 mrx.py -d target.com



خيارات إضافية

Bash


# تحديد مجلد مخرج معين
python3 mrx.py -d target.com -o my_scan_results

# عرض المساعدة
python3 mrx.py --help






📂 هيكل النتائج

بعد انتهاء الفحص، ستجد مجلد النتائج يحتوي على:

•
subs.txt: قائمة النطاقات الفرعية.

•
live.txt: المواقع التي تعمل فعلياً.

•
nuclei_results.txt: الثغرات المكتشفة بواسطة Nuclei.

•
params_*.txt: الروابط المصنفة حسب نوع الثغرة.

•
sqlmap_results/: نتائج فحص حقن قواعد البيانات.
